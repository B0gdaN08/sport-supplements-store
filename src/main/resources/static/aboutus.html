The Project

SportSupps Store is a university project developed for the Distributed Systems course. It simulates a real e-commerce platform where users can browse supplements, manage categories and place orders through a REST API.

Features
  • Product, category and order management
  • Full CRUD operations
  • REST API architecture
  • User authentication and roles
  • Frontend connected to backend services
Technologies

Frontend: HTML, CSS, JavaScript

Backend: Java (Spring Boot)

Architecture: REST API

Storage: In-memory repository
